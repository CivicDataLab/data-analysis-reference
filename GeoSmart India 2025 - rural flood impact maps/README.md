This repository contains data, code, notebooks and map outputs used to produce district and subdistrict level rural flood impact maps. This analysis was used for GeoSmart India 2025.

- the folder "layers" contains vector and delimited layers used to generate maps in QGIS
- exports contains output images
- scripts contains python scripts used to concatenate and transform data for analysis

to extract bhuvan flood hazard map, use the following link (add as WMS/WMTS tiles layer): https://bhuvan-ras2.nrsc.gov.in/cgi-bin/hazard.exe
